# 🎮 Tic Tac Toe Game (Java)

A simple console-based **Tic Tac Toe** game developed using Java.
This project allows a human player to play against an AI opponent.

---

## 🚀 Features

* 🧑‍💻 Human vs AI gameplay
* ✅ Input validation (prevents invalid moves)
* 🔄 Turn-based system
* 🏆 Win detection (rows, columns, diagonals)
* 🤝 Draw detection

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Abstraction, Inheritance, Polymorphism)
* Eclipse IDE

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/TicTacToe-Java.git

2. Open in Eclipse or any Java IDE

3. Run:
   LaunchGame.java

---

## 🎯 How to Play

* Enter row and column values (0 to 2)
* Player uses 'X'
* AI uses 'O'
* First to align 3 marks wins
